## 🚀 Featured Projects

Here is a curated showcase of my top work, ranging from edge AI software to web systems built during hackathons:

### 📡 [RuView: WiFi CSI Gesture & Vitals Sensing](https://github.com/sahilalamsiddik/wall_view)
*Camera-free occupancy, vital signs, and human pose tracking using standard WiFi signals.*
- **Features**: Utilizes ESP32 Channel State Information (CSI) streams, processed via Spiking Neural Networks (SNNs) in Rust, rendering a 3D hologram observatory in Three.js over WebSockets.
- **Tech Stack**: `Rust` • `Node.js` • `WebAssembly` • `Tauri v2` • `Docker` • `Three.js`

### 🛡️ [SafeSpeak: Community Safety Platform](https://github.com/sahilalamsiddik/SafeSpeak_GDG_hackathon)
*GDG Hackathon project enabling anonymous public safety reporting with active dashboard routing.*
- **Features**: Animated React 19 UI with Framer Motion, customized REST APIs mapping locations to an incident triage board, and status management workflows.
- **Tech Stack**: `React 19` • `Node.js` • `Express` • `MongoDB` • `Tailwind CSS` • `Framer Motion`

### 💬 [MeshChat: Offline WiFi Mesh Messenger](https://github.com/sahilalamsiddik/offline_whatsapp)
*A local group messenger application modeled after WhatsApp that operates entirely without cellular signals or internet.*
- **Features**: Establishes a local WebSocket message broker via a local host Node.js server, allowing real-time chat rooms to function across local WiFi routers.
- **Tech Stack**: `Node.js` • `Socket.io` • `Express` • `HTML5` • `JavaScript`

### 🌱 [Smart Irrigation System: Telemetry & IoT Dashboard](https://github.com/sahilalamsiddik/HACKZION.V3__Event-Horizon)
*Cyber-physical plant monitor streaming real-time soil moisture and environmental metrics.*
- **Features**: ESP8266 NodeMCU sensor polling (capacitive soil moisture & DHT22), streaming structured telemetry arrays via HTTP client loops to a live React dashboard.
- **Tech Stack**: `C++` • `ESP8266` • `Arduino IDE` • `React.js` • `Node.js`
